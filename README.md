# CS Book Library

A personal Computer Science book library built with HTML and CSS.

## Features

- 📚 24 classic and modern CS books organised into 6 categories
- 🔍 Live search by title or author
- 🏷️ Filter by category (Algorithms, Programming, Systems, Software Engineering, Theory, AI/ML)
- 🎨 Reading-status badges: **Read**, **Currently Reading**, **To Read**
- 📱 Fully responsive layout

## Categories

| Category | Colour |
|---|---|
| Algorithms & Data Structures | Blue |
| Programming | Green |
| Systems | Yellow |
| Software Engineering | Purple |
| Computer Science Theory | Red |
| AI & Machine Learning | Orange |

## Usage

Open `index.html` in any modern browser — no build step required.
